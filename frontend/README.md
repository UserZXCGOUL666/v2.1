# Storefront

Клиентская часть магазина. Разворачивается отдельным проектом Vercel из папки `frontend`.

Переменные:

```env
VITE_USE_MOCK=false
VITE_API_URL=http://localhost:3000
VITE_ADMIN_URL=http://localhost:5174
VITE_SUPPORT_BOT_URL=https://t.me/your_support_bot
```

Команды:

```bash
npm install
npm run dev
npm run build
```
